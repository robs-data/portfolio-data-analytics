# 🚗 Projeto 01 - Locadora de Veículos

## 📌 Objetivo

Desenvolver um dashboard para acompanhar os indicadores da locadora de veículos, auxiliando a tomada de decisões estratégicas.

---

## 📂 Fonte dos Dados

Os dados foram disponibilizados em planilhas Excel contendo informações sobre:

- Clientes
- Veículos
- Locações
- Faturamento

---

## 🛠 Tecnologias Utilizadas

- Power BI
- SQL
- Python
- Excel

---

## 🧹 Limpeza dos Dados

Foram realizadas as seguintes etapas:

- Remoção de duplicidades
- Tratamento de valores nulos
- Padronização de datas
- Correção de tipos de dados
- Criação de colunas auxiliares

---

## 📊 Análise dos Dados

As principais análises desenvolvidas foram:

- Receita mensal
- Receita anual
- Clientes ativos
- Clientes inativos
- Veículos mais alugados
- Ticket médio
- Evolução das locações

---

## 📈 Resultados

O projeto permitiu identificar:

- Veículos mais rentáveis
- Períodos com maior faturamento
- Perfil dos clientes
- Tendências de crescimento
